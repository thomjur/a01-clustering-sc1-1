{ وَلَقَدْ مَنَنَّا عَلَىٰ مُوسَىٰ وَهَارُونَ } يريد أعطينا موسى وهارون { وَنَجَّيْنَاهُمَا وَقَوْمَهُمَا } يريد بني إسرائيل الاثنى عشر سبطا { مِنَ ٱلْكَرْبِ ٱلْعَظِيمِ } يريد الظلم العظيم { وَنَصَرْنَاهُمْ فَكَانُواْ هُمُ ٱلْغَٰلِبِينَ } يريد لفرعون { وَآتَيْنَاهُمَا ٱلْكِتَابَ ٱلْمُسْتَبِينَ } يريد التوراة وما فيها من الأحكام { وَهَدَيْنَاهُمَا } يريد أرشدناهما { ٱلصِّرَاطَ ٱلْمُسْتَقِيمَ } يريد الدين القويم الواضح { وَتَرَكْنَا عَلَيْهِمَا فِي ٱلآخِرِينَ } يريد الثناء الحسن { سَلاَمٌ عَلَىٰ مُوسَىٰ وَهَارُونَ } . { إِنَّا كَذَلِكَ نَجْزِي ٱلْمُحْسِنِينَ } يريد الموحدين { إِنَّهُمَا مِنْ عِبَادِنَا ٱلْمُؤْمِنِينَ } يريد المصدقين بتوحيد الله .
