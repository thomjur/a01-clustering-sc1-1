{ وَقَالُواْ لَوْلاَ } هلا { يَأْتِينَا بِآيَةٍ مِّن رَّبِّهِ } قال الله { أَوَلَمْ تَأْتِهِمْ بَيِّنَةُ } قال محمد يعني آيات { مَا فِي ٱلصُّحُفِ ٱلأُولَىٰ } يعني التوراة والإنجيل { وَلَوْ أَنَّآ أَهْلَكْنَاهُمْ بِعَذَابٍ مِّن قَبْلِهِ } يعني من قبل القرآن { لَقَالُواْ رَبَّنَا لَوْلاۤ } هلا { أَرْسَلْتَ إِلَيْنَا رَسُولاً } . { قُلْ كُلٌّ مُّتَرَبِّصٌ } نحن وأنتم كان المشركين يتربصون بالنبي أن يموت وكان النبي يتربص بهم أن يجيئهم العذاب { فَسَتَعْلَمُونَ مَنْ أَصْحَابُ ٱلصِّرَاطِ ٱلسَّوِيِّ } يعني الطريق المعتدل { وَمَنِ ٱهْتَدَىٰ } أي فستعلمون أن النبي والمؤمنين كانوا على الصراط السوي وأنهم ماتوا على الهدى .
