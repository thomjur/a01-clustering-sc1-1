ثم قال عز وجل { وَأْمُرْ أَهْلَكَ بِٱلصَّلوٰةِ } يعني قومك وأهلك وأهل بيتك بالصلاة { وَٱصْطَبِرْ عَلَيْهَا } يعني : اصبر على ما أصابك فيها من الشدة روى عبد الرزاق عن معمر عن رجل : أن النبي - صلى الله عليه وسلم - كان إذا دخل عليه نقص في الرزق أي ضيق أمر أهله بالصلاة . ثم قرأ { وَأْمُرْ أَهْلَكَ بِٱلصَّلوٰةِ وَٱصْطَبِرْ عَلَيْهَا } { لاَ نَسْأَلُكَ رِزْقاً } لخلقنا ولا أن ترزق نفسك إنما نسألك العبادة { نَّحْنُ نَرْزُقُكَ } في الدنيا ما دمت حياً { وَٱلْعَـٰقِبَةُ لِلتَّقْوَىٰ } يعني : الجنة للمتقين { وَقَالُواْ } يعني الكفار { لَوْلاَ يَأْتِينَا بِـئَايَةٍ مّن رَّبّهِ } يعني : هلا يأتينا محمد بعلامة لنبوته قال الله تعالى { أَوَ لَمْ تَأْتِهِمْ بَيّنَةُ } يعني : بيان { مَا فِى ٱلصُّحُفِ ٱلأُولَىٰ } يعني ما في التوراة والإنجيل حتى يجدوا نعته فيه وهذا كقوله عز وجل { فَاسْأَلِ ٱلَّذِينَ يَقْرَءُونَ ٱلْكِتَـٰبَ مِن قَبْلِكَ } [ يونس : 94 ] ثم قال عز وجل : { وَلَوْ أَنَّا أَهْلَكْنَـٰهُمْ بِعَذَابٍ مّن قَبْلِهِ } يقول : لو أن أهل مكة أهلكناهم قبل محمد - صلى الله عليه وسلم - والقرآن { لَقَالُواْ رَبَّنَا لَوْلا أَرْسَلْتَ إِلَيْنَا رَسُولاً فَنَتَّبِعَ ءايَـٰتِكَ مِن قَبْلِ أَن نَّذِلَّ وَنَخْزَىٰ } يعني : من قبل أن نعذب ثم قال عز وجل : { قُلْ كُلٌّ مُّتَرَبّصٌ } يعني : منتظر لهلاك صاحبه أنا وأنتم وقال مقاتل : كان كفار مكة يقولون نتربص بمحمد { رَيْبَ ٱلْمَنُونِ } [ الطور : 30 ] يعني : الموت ووعدهم النبي - صلى الله عليه وسلم - العذاب فأنزل الله تعالى { قُلْ كُلٌّ مُّتَرَبّصٌ } يعني : أنتم متربصون بمحمد - صلى الله عليه وسلم - الموت ومحمد متربص بكم العذاب فأنزل الله تعالى { قُلْ كُلٌّ مُّتَرَبّصٌ } { فَتَرَبَّصُواْ } أي : انتظروا { فَسَتَعْلَمُونَ } إذا نزل بكم العذاب { مَنْ أَصْحَـٰبُ ٱلصّرَاطِ ٱلسَّوِيّ } أي العدل { وَمَنِ ٱهْتَدَىٰ } منا ومنكم قرأ نافع وأبو عمرو وعاصم ( أَوَلَمْ تَأْتِهِمْ ) بالتاء لأن لفظ البينة مؤنث والباقون أولم يأتهم بالياء لأن معناه البيان والله سبحانه وتعالى أعلم وصلى الله على سيدنا محمد وعلى آله وصحبه وسلم .
