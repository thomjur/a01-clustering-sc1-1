ثم استأنف وقال : { وَهَل أَتَاكَ نَبَؤُاْ الْخَصْمِ } أي : خبر الخصم ، أي : إِنك لم تعلمه حتى أعلمتك . { إِذْ تَسَوَّرُواْ الْمِحْرَابَ } أي : المسجد . { إِذْ دَخَلُواْ عَلَى دَاوُودَ فَفَزِعَ مِنْهُمْ قَالُواْ لاَ تَخَفْ خَصْمَانِ بَغَى بَعْضُنَا عَلَى بَعْضٍ فَاحْكُم بَيْنَنَا بِالْحَقِّ وَلاَ تُشْطِطْ } أي : ولا تَجُر { وَاهْدِنَا } اي : وأرشدنا { إِلَى سَوَآءِ الصِّراطِ } . أي : إلى قصد الحق . وقال بعضهم : إلى سواء الصراط ، أي : إِلى عدل القضاء . { إِنَّ هذَآ أَخِي لَهُ تِسْعٌ وَتِسْعُونَ نَعْجَةً } أي : امرأة { وَلِيَ نَعْجَةٌ وَاحِدَةٌ فَقَالَ أَكْفِلْنِيهَا } [ أي انزل عنها لي وضُمَّها إليّ ] . { وَعَزَّنِي } أي : وغلبني وقهرني . وهي تقرأ على وجه آخر : وعازني { فِي الْخِطَابِ } وتفسير ذلك بعد هذا الموضع .
