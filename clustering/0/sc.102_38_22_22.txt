{ إِذْ دَخَلُواْ عَلَىٰ دَاوُودَ فَفَزِعَ مِنْهُمْ قَالُواْ لاَ تَخَفْ } أي : منا . فلسنا فاتكين وإنما نحن { خَصْمَانِ } أي : شخصان متخاصمان تحاكمنا إليك { بَغَىٰ بَعْضُنَا عَلَىٰ بَعْضٍ } أي : تعدى { فَٱحْكُمْ بَيْنَنَا بِٱلْحَقِّ } أي : بما يطابق أمر الله { وَلاَ تُشْطِطْ } أي : ولا تبعد عن الحق أو تجاوزه { وَٱهْدِنَآ إِلَىٰ سَوَآءِ ٱلصِّرَاطِ } أي : بحيث لا تميل عن الحق أصلاً .
