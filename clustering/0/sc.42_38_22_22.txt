{ إِذْ دَخَلُواْ عَلَىٰ دَاوُودَ فَفَزِعَ مِنْهُمْ } لانّهم دخلوا فى غير وقت دخول الاغيار ودخلوا من دون اذن ومن غير المحلّ المعتاد للدّخول { قَالُواْ } بعد ما رأوا انّه فزع منهم { لاَ تَخَفْ خَصْمَانِ } كأنّهم كانوا جماعةً وقال بعضهم : هذان خصمان ، او : نحن خصمان { بَغَىٰ بَعْضُنَا عَلَىٰ بَعْضٍ فَٱحْكُمْ بَيْنَنَا بِٱلْحَقِّ وَلاَ تُشْطِطْ } لا تجر فى الحكومة { وَٱهْدِنَآ إِلَىٰ سَوَآءِ ٱلصِّرَاطِ } المرضىّ لله وللعقل .
